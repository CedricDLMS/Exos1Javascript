###Exercice 1

## Description des fichiers

- **classes/**
  - `form.js`: Ce fichier contient la définition de la classe `Form`, qui gère la création et la manipulation de formulaires.

- **tools/**
  - `addCss.js`: Ce fichier contient des fonctions pour ajouter dynamiquement des fichiers CSS.
  - `Create.js`: Ce fichier contient des fonctions utilitaires pour créer des éléments HTML de manière dynamique.

- `Constants.js`: Ce fichier contient des constantes utilisées dans le projet.
- `index.html`: Le fichier HTML principal qui charge l'application.
- `index.js`: Le fichier JavaScript principal qui initialise et gère les interactions de l'application.
- `README.md`: Ce fichier, contenant la description du projet.


###Exercice 2 

## Description des fichiers

- **classes/**
  - `select-commune-final.js`, `select-communes.js`, `select-departement.js`, `select-region.js`: Ces fichiers contiennent des classes pour gérer les sélections de communes, départements et régions.

- **tools/**
  - `Create.js`: Contient des fonctions utilitaires pour créer des éléments HTML.
  - `DepartementFetch.js`, `FetchCommuneFinal.js`, `FetchCommunes.js`, `RegionsFetch.js`: Contiennent des fonctions pour récupérer des données via des appels API.

- `index.html`: Le fichier HTML principal qui charge l'application.
- `index.js`: Le fichier JavaScript principal qui initialise et gère les interactions de l'application.
- `styles.css`: Le fichier de styles CSS.
